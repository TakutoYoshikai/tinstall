# tinstall
tinstall is a script to install Takuto Yoshikai's open source software.

### Requirements
* macOS or Ubuntu

### Supports Takuto Yoshikai's
* pip packages
* npm packages
* cargo packages
* shell scripts


### Usage

**install**
```bash
npm install -g TakutoYoshikai/tinstall
```

**install software**
```bash
tinstall <repository name>
```
